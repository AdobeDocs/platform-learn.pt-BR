# Consultar e descobrir dados no Data Science Workspace

O Adobe Experience Platform permite usar a Linguagem de consulta estruturada (SQL) no Data Science Workspace ao integrar o Serviço de consulta ao JupyterLab como um recurso padrão. Este tutorial demonstra alguns casos de uso comuns para explorar, transformar e analisar dados. Para obter mais informações, visite o [Serviço de consulta na documentação do Data Science Workspace](https://experienceleague.adobe.com/docs/experience-platform/data-science-workspace/jupyterlab/query-service.html).
