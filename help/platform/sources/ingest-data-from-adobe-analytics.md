# Assimilar dados usando o conector de origem do Adobe Analytics

O conector de origem da Adobe Analytics permite fazer stream, mapear e filtrar facilmente dados da Adobe Analytics para o lago de dados do Perfil do cliente em tempo real e da experiência da Adobe Experience Platform. Saiba como usar os recursos de preparação de dados para criar campos semânticos e calculados para melhorar a experiência do usuário no Construtor de segmentos e nos aplicativos da Plataforma. Conjuntos de relatórios entre regiões são compatíveis com esse conector. Para obter a documentação mais detalhada do produto, consulte [Criar um conector de origem do Adobe Analytics na interface do usuário](https://experienceleague.adobe.com/docs/experience-platform/sources/ui-tutorials/create/adobe-applications/analytics.html?lang=pt-BR).
