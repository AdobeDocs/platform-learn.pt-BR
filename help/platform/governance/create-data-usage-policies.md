# Criar políticas de uso de dados

Saiba como criar e gerenciar políticas de uso de dados na interface do usuário do Adobe Experience Platform. As políticas de uso de dados permitem restringir o uso de dados para fins específicos com base na rotulagem de dados. Para saber mais sobre a abordagem baseada em API, consulte [criar políticas de uso de dados com a API](https://experienceleague.adobe.com/docs/experience-platform/data-governance/policies/create.html).
