# Demonstração do Real-time Customer Data Platform

Este vídeo mostra como o Real-Time CDP é usado para coletar dados de várias fontes, mesclar esses dados em um único perfil do cliente em tempo real e ativá-los para criar experiências do cliente personalizadas. Para obter mais informações, visite o [Documentação do Real-time Customer Data Platform](https://experienceleague.adobe.com/docs/experience-platform/rtcdp/overview.html?lang=pt-BR).
