# Usar a interface do Adobe Experience Platform Agent Orchestrator

Saiba como interagir com soluções da Experience Cloud por meio de prompts contextuais, objetos dinâmicos e vários agentes especializados usando o Adobe Experience Platform Agent Orchestrator. O Agent Orchestrator fornece um fluxo de trabalho transparente ao expor a lógica de raciocínio, as consultas SQL e o histórico de conversas. Para obter mais informações, consulte a [documentação do Agent Orchestrator](https://experienceleague.adobe.com/pt-br/docs/experience-cloud-ai/experience-cloud-ai/agents/agent-orchestrator).
