# Analytics e agentes

## Objetivos de aprendizagem

Neste módulo, você analisará dados que residem no Adobe Analytics e/ou Customer Journey Analytics usando agentes externos que não sejam da Adobe.

## Pré-requisitos

- Acesso ao Adobe Analytics (AA) e/ou Adobe Customer Journey Analytics (CJA)
- Acesso ao ChatGPT e/ou Claude.ai com assinatura paga

## Exercícios

[1.5.1 CJA e ChatGPT com servidor MCP](./ex1.md){target="_blank"}

Neste exercício, você configurará um servidor MCP personalizado no ChatGPT para conectar o ChatGPT diretamente ao Customer Journey Analytics e começará a analisar seus dados hospedados no Adobe Customer Journey Analytics (CJA) diretamente no ChatGPT.

[1.5.2 CJA e Claude.ai com servidor MCP](./ex2.md){target="_blank"}

Neste exercício, você configurará um servidor MCP personalizado no Claude.ai para conectar o Claude.ai diretamente ao Customer Journey Analytics e começará a analisar seus dados hospedados no Adobe Customer Journey Analytics (CJA) diretamente do Claude.ai.

[1.5.3 Adobe Analytics e Claude.ai com servidor MCP](./ex3.md){target="_blank"}

Neste exercício, você configurará um servidor MCP personalizado no Claude.ai para conectar o Claude.ai diretamente ao Adobe Analytics e começará a analisar seus dados hospedados no Adobe Analytics (AA) diretamente do Claude.ai.
