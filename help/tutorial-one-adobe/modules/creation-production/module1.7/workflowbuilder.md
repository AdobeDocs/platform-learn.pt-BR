# 1.7 Firefly Creative Production for Enterprise

O foco principal da Firefly Creative Production for Enterprise são os tecnólogos criativos e os construtores de soluções que criam e configuram fluxos de trabalho.

Com o Firefly Creative Production for Enterprise, você pode:

- Criar fluxos de trabalho: crie fluxos de trabalho de produção em várias etapas usando uma interface visual com assistência manual ou por IA.
- Executar fluxos de trabalho: executa fluxos de trabalho em escala, incluindo processamento em lote, com um tempo de execução robusto que opera fora do navegador.
- Implantar fluxos de trabalho: publique fluxos de trabalho como endpoints da API ou interfaces programáticas do usuário para integração em outras ferramentas e sistemas.
- Integrar fluxos de trabalho: conecte fluxos de trabalho a sistemas corporativos, como DAMs, fluxos de ativação e cadeias de fornecimento de conteúdo.

Os principais casos de uso compatíveis incluem itens como:

- Adaptações do Canal de mídia social, como a criação de variantes de tamanho para cada plataforma e tipo de mídia social.
- Preparação de ativos, como preparar ativos para uso em superfícies adicionais, como impressão ou Web.
- Redirecionamento de conteúdo, como atualização e reutilização de recursos visuais para reengajar públicos-alvo interessados.
- Comércio eletrônico, como normalização de imagens de produtos e criação de variantes específicas de segmento para públicos diferentes.

## Pré-requisitos
